A breadcrumb plugin for Elementor that uses the functions of The SEO Framework.

== Description ==

Breadcrumbs For Elementor is a powerful yet easy-to-use plugin that adds a fully customizable breadcrumbs widget to your Elementor page builder. This plugin leverages the functionality of The SEO Framework to provide efficient and lightweight breadcrumbs for your website.

Features:

* Seamless integration with Elementor
* Utilizes functions of The SEO Framework plugin
* Choose if you want to display Homepage and under which name
* Choice between icon or text dividers
* Fully responsive design
* Extensive styling options (colors, typography, spacing, alignment)
* SEO-friendly markup
* Accessibility-ready with proper ARIA labels

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/breadcrumbs-for-elementor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Ensure that Elementor and The SEO Framework plugins are installed and activated.
4. Use the Elementor editor to add the 'Breadcrumbs' widget to your pages.

== Frequently Asked Questions ==

= Do I need Elementor Pro to use this plugin? =

No, Breadcrumbs For Elementor works with both the free and pro versions of Elementor.

= Can I customize the appearance of the breadcrumbs? =

Yes, the plugin offers extensive customization options. You can change colors, typography, spacing, alignment, and more directly in the Elementor editor.

= Why does this plugin use The SEO Framework? =

It is a great plugin. That is why we used there functions for generating the breadcrumbs. If you have The SEO Framework activated
this plugin won't load the included functions but will use the functions directly from The SEO  Framework.
